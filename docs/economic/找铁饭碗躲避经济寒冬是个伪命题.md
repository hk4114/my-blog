# 为什么我说“找铁饭碗躲避经济寒冬”是个伪命题？

在这个秋招季，很多年轻人感到焦虑，大家都倾向于追求稳定。中国人骨子里似乎都在追求安全感、铁饭碗和安逸。从儒家思想道科举制度，再到如今对体制内编制的追逐，几千年的经验似乎都在告诉我们：“稳定等于成功，确定性等于幸福。”

但我必须泼一盆冷水：当整个社会都在追求确定性时，确定性反而成了最稀缺的奢侈品。 盲目追求“铁饭碗”来躲避经济寒冬，不仅代价高昂，甚至可能是一个彻头彻尾的伪命题。

## 确定性是一种昂贵的“溢价”

为什么华为的手机有时比苹果卖得贵？并非华为一定做得比苹果好，而是因为不确定性本身就是一种成本。

苹果拥有全球公认的最具“确定性”的供应链体系。作为供应商，如果你能进入苹果的体系，你面临的是严格的标准、稳定的现金流和长期的订单。因为这种确定性，供应商愿意在价格上让步。这就是“批发”与“零售”的区别：当你能提供可预测的收入、持续的需求和长期的关系时，对方就愿意降低价格。

反之，如果不确定性高，价格必然包含风险溢价。这也是为什么在经济学逻辑里，追求极致的稳定往往需要付出极大的代价——也就是现在大家感受到的“极度内卷”。

## 公务员薪资的经济学本质：类国债收益

很多人认为公务员是“铁饭碗”，是因为它提供了一种确定性。但从经济学角度看，公务员的工资收益曲线，其实非常像国债。

- 国债利率：是国家借钱的成本，反映政府融资价格。
- 公务员工资：是国家支出的组成部分，反映政府雇佣劳动力的成本。

两者的资金来源都是财政预算。当经济景气、通胀预期强时，税收增加，财政收入上升，国债收益率高，公务员的工资和福利也随之水涨船高。但是，从 2020 年到 2023 年，我们可以看到国债收益率跌破 3%，宏观经济下行，随之而来的就是公务员绩效的延缓发放。

所以，所谓的“稳定”并不独立于宏观经济而存在。

## 为什么说“避险”是伪命题？

我一直认为，指望通过考公来躲避经济寒冬是一个伪命题。

经济下行时，寒气是一尺一尺冻下去的。最先倒霉的是那些身先士卒冲入市场的人（企业、个体户），但体制内并不是避风港，薪资政策的调控只是具有滞后性而已。大家都是一条绳上的蚂蚱，你指望市场没钱的时候，财政能独善其身给你发高薪吗？

如果宏观经济未来几年不能好转，那些大家趋之若鹜的体制内职位，很可能会变成“又臭、又硬、又累、又卷”且工资低的天坑岗位。

这时候再去考公的人，处境会非常尴尬。因为体制内的体面薪水是建立在经济繁荣之上的。只有当宏观经济好转，“春江水暖鸭先知”的市场先行者赚得盆满钵满时，体制内的工资才会看起来稍微体面一点。

## 只有这三种人适合进体制

如果你不是为了避险，而是属于以下三种情况，进体制是可以理解的：

1. 有资源、能爬很高的人：这类人进体制本身就不是为了“避险”，而是为了权力和更高的上限。
2. 极度贪婪或拥抱高风险的人：这类人想通过非正常手段获利（当然这是违法的，且逻辑与避险相反）。
3. 真正热爱这份工作的人：就像我做自媒体，哪怕收入减少十倍我也愿意直播，因为热爱高于一切。

除了这三种情况，普通人抱着“混日子”或“避险”的心态进去，大概率会失望。

## 逆向思维：熊市才是奋斗的好时候

现在大家都在喊“躺平”，但我却觉得现在是“卷”的好时候。

- 牛市（经济好时）：市场非常吵闹，泡沫巨大，“乱花渐欲迷人眼”。你要在 100 个垃圾项目中找 1 个有价值的东西，成本极高。很多糟糕的项目仅仅因为市场有钱就能假模假样地运转，财报极其漂亮，但长期看并不稳健。
- 熊市（经济差时）：潮水退去，才知道谁在裸泳。市场上到处是“带血的筹码”，资产被低价甩卖，信任比金钱更稀缺。

大众的逻辑往往是错的： 别人卷的时候你也卷，别人躺的时候你也躺。
我的逻辑是： 在牛市大家狂欢时，我倾向于躺平（因为机会成本高、泡沫大）；在熊市大家躺平时，我反而更愿意工作和投资。

### 我的投资案例

- 关于黄金：正是因为局势紧张、经济不好，我才买黄金。
- 关于腾讯：当时很多人因为腾讯出海做得不好而不看好它。我反而认为，正因为出海不强势，它在复杂的地缘政治和海外局势紧张时，受到的摩擦和制裁风险反而更小。这叫“塞翁失马，焉知非福”。

在熊市里赚的钱，往往比牛市里多得多，而且更扎实。因为这时候你能买到没有泡沫的资产。

## 结语：拒绝赌徒心态

虽然我鼓励在熊市奋斗，但绝不是鼓励去赌。

经济有周期，钱通常不会消失，只会转移——从一个人手里流向另一个人。不要去做高风险的“赌狗”，尤其是在币圈或高杠杆投资中，很多人因为回撤太大，一波就被清空了。我见过朋友从 A9（九位数资产）亏到 A7，非常惨痛。

好好打磨自己的认知，做有把握的事情。只要在这个“比烂”的时代里，你稍微积极一点，认知稍微深一点，你就已经领先了 99%的人。

与其在体制内的幻象中寻求虚假的安逸，不如在遍地黄金的“废墟”中寻找真正的机会。
