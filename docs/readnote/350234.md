# 你真的看懂 350234 了吗？

很多人将其总结为苦命鸳鸳，但在我看来，作者想表达的感受远不止如此。1000 个人眼中有 1000 个哈姆雷特，一部好的作品自然值得我们细细地去品读。
首先关于董卓。不管是偏正史的《三国志》还是小说《三国演义》，对他的描写都是一个暴君形象：董卓进京废立皇帝，秽乱后宫，屠杀百官。因为自己算不得正统皇室，无法名正言顺地坐上帝王之位，所以就用这种方式让天下来惧怕他、来臣服他。
而这部漫画作者的厉害之处，就是能够抓住历史中被人忽视的细节。作者深知，一个人无论表现得再怎么冷酷，其内依然是有血有肉的，是温暖的。在作者看来，董卓即使生于乱世，也必然有着柔情的一面，这从董卓对当世英雄的态度就能看得见：曹操、荀攸之辈谋害董卓失败，董卓发泄后对其的处置不是当场处死，而是囚禁，望英雄能为自己所用。
那作者的故事为何不从此处展开，更加立体地表达董卓对当世英雄的惺惺相惜呢？因为求也得排队，当然更是因为奉先之辈对董卓而言有着非常重要的意义。
董卓与吕布相遇后，不惜以重金求之；吕布也深感知遇之恩，为董卓诛五俘，指孟德，战三英，多次救董卓于水火之中。由此为基础，作者将故事情节一步步推进到情投意合、互诉衷肠，就显得水到渠成：既是君臣，又是父子，亦是鸳鸳，忠孝即全，令人瞠目。
而且作者对二人感情的描写，并非像董卓对其他英雄那样冰释前嫌——因为那段感情太过于浅显而经不起考验。作者采用的是更加复杂、更加立体的表现手法：在文笔上是高山流水觅知音的酣畅，是一见如故的喜悦，是怦然心动的沉醉；在内核上，是对古代君臣父子体系内“君臣”本身特有的韵律气质的表现。用克苏鲁级别的画风掩盖作品中极具深意的内核，由此可见作者文笔之深厚，胆大且心细。
这还只是前期的铺垫。这部漫画真正的动人之处在于乱世悲情的收束：董卓被俘，而吕布则是奉旨奉诏讨贼成功的英雄，二人原本的身份就此支离破碎。念于旧情，吕布不愿再看董卓受尽折磨，便亲手将其送上刑场；在欢呼声中，亲手将自己往日的种种撕碎，化作众人狂欢的燃料。
最后董卓留于吕布的温存见识，犹如人类最为原始的情感：触碰时的温暖和分离时的永诀。吕布的反应与所处环境形成强烈的反差，以乐景衬哀情，营造出了一种极为矛盾的氛围，强化了结尾的情感张力。
同许多伟大的文学作品一样，这部漫画将矛盾留在了结尾。即使我们知道是虚构的，但这种矛盾给人的情感却十分的现实，仿佛在问读者：可否记得往日种种？还同时真的想成为科学家和宇航员吗？那 800 字作文的内容真的是心中所想吗？所填志愿真的是心中所愿吗？所找工作真的是心中所爱？
每个人都是被人架起的吕布。
“狂欢的众人说道，这是正确的。”
“狂欢的众人喊道，这是幸福的。”
狂欢的众人是不存在的，但他们的声音却震耳欲聋。
同时每个人又是被往日所困的董卓。生活问了每个人，可还有话说？
不是不说，而是无法实现的未来只能化作插在自己心上的刀：拔出来是血，不拔出来是疼。
我们有太多无法自己左右的事，董卓、吕布是如此，更何况我们平凡人呢？
董卓与吕布的关系就像山与林一样：没有树木的山，毫无生气；350234，山无林而山死，林立于山间，为原来沉寂的山脉带去了生机；树木也借由山的高度看得更远。
同时山自己也明白，树林一旦茂密，山本来的样貌便会在人们眼中逝去，也就是“山悟林而山逝”。
董卓无吕布无以成大业，此乃“山无林而山死”；董卓知晓吕布的难处，便甘愿逝去，此乃“山悟林而山逝”。这便是 350234 这段编号所蕴含的深意。
那山之后是什么呢？作者借吕布给出了自己的答案：每个人都有自己的山，越过去，自己看。
