# 从最后通牒博弈看经济信心与青年困境

在行为经济学中，有一个最后通牒博弈理论。即AB两个参与者，A是提议者，B是接受者。如果有100元要分配，A可以提议各自分配一半，也可以自己分配多一点，然后由B来决定接受不接受。理论上作为一个理性经纪人，不管A给B分配多少，哪怕只分配一块钱，对于B来说也是完全新增的，所以看起来B应该怎么样都会答应。但实验结果显示，B对A所提出的不公平的分配会予以否决。比如A提议分配给B20%以下的分配比例，B就会否决，而尽管这样，B会损失掉净增20%的利益，他们也要阻止A能轻松获得80%利益的不公平行为，这就是心理学上的消极互惠。

我们常常在新闻中看到有人拿着ID实名举报，而尽管这样，他本人承担着很多风险和成本，他们也想要让犯错的人接受相应的惩罚，就是这个道理。话说回来，前面谈到这新增的100元在真实经济运行中也并非完全凭空创造的，它借债务的名义还是让B承担了经济成本，所以B更加不会愿意A在分配上过于偏袒自己，而一旦被发现就会产生消极互惠的念头。这也就是为什么A会竭力隐藏自己的分配优势和各种数据，但隐藏本身就会产生另一种不信任。

我们想象在实验中，B不知道总额是不是100元，只被告知自己可以被分配20元，那么B就会产生猜疑，而A被分配的金额最终会在使用环节被察觉而进而加深猜疑，其影响也不可小觑。比如明明总额是100元，却被猜疑成200元，显然猜疑的成本有时候会更大。最好的降低成本的方式就是透明，然后倾向于B可以看到，然后就会从消极转为积极。而这玩意说白了其实很简单，也不需要经济学家反复对着每月的数据念经，然后给出各种荒了大唐的建议。毕竟天地之间有杆秤，那秤砣宏观需要思考下，当居民部门不贷款抗税，是不是本来就不该用这个来征税和填充GDP呢。

那为什么中国的家长如此重视教育呢？很简单，因为能考上一所好学校，意味着未来能有一份好工作的概率提升，也意味着孩子长大之后在社会上能有更好的机会、选择权和收入增长。它背后折射的是社会环境的一种大众默契，就是好工作等于好生活，而现实是好工作是稀缺的，所以竞争好工作的成本是必须的，而无论是金钱成本还是孩子的身心成本。

那么在国内什么属于好工作呢？他又为什么稀缺呢？我们以统计年鉴中的平均工资为标准，可以看到2023年的数据，从最大分类来看，非私营单位的平均工资远高于私营单位平均工资，差不多是1.8倍，而非私营单位的就业人数只有私营单位就业人数的一半左右。所以不难想象，很多人的首选就业对象就是非私营单位，而尤其是其中的体制内、国企、外资三类。而根据2023年的公积金年报，我们可以看到机关事业单位和国企的员工总数相加大概在8100万，这不含编外人员，而三资企业加起来员工数只有2000万。而由于这几年外部环境发生变化，不少外资企业要么降本裁员，要么索性撤资，所以这一块的就业岗位是有所缩减的。比如从2016年到2023年，我算了一下三资企业的就业规模总共少了600万左右。所以在非私营单位中，一直被称为铁饭碗、工作稳定的体制及国企岗位，成为了众多毕业生的首选。这个可参考智联招聘的毕业生调查报告，而近3/4的受访者第一选择为机关事业单位或国企，从侧面说明上述平均工资数据在毕业生择业中的分量。这个第一选择的就业规模实际增长了742万，而我们再想想同应届毕业生有多少，总计7000多万，也就是大部分人其实无法如愿，而但在结果出来前，他们都会尽力去尝试去内卷。

那么大家为什么去私营企业或者其他内资企业的意愿那么低呢？这是因为这类企业的类型的雇佣对求职者来说相对不稳定，而不仅需要更卷的去投入自己的时间精力去竞争，而即使在大厂也是如此，而且还面临着35岁被优化的担忧。这从城镇员工的劳动时间变化可以看到，2015年大家一周只要工作45.5小时，而到了2023年，大家却要工作49小时。无论这种劳动时间拉长是资源的还是被迫的，都是为了保证自己的就业饭碗，因为钱数无法获得稀缺资源的剩余，新增劳动力每年都在大量冲入就业市场，大家都有危机感。

那么为什么经济运行不能提供更多的就业岗位出来，让大家不要那么内卷呢？而这就和利润分配有关了。实际收入不变的情况下，商品价格的变化就会引起替代品的需求量。比如现在很多按摩店涨价，更多人开始选择按摩产品，像我数次推荐的妙界肩颈按摩仪已经售出百万份，而销量稳居按摩类top one。很多人反馈，他复刻了传统老师傅的手法，缓解疲累，舒筋活络效果绝佳。我用了一年多，也明显感觉肩背轻松很多，它的按摩头是我见过最像人手的配置，手法要比同类产品惊艳的多，真正能揉捏僵直的脖颈，推开紧绷的斜方肌，十几分钟就可以把每一处淤结都按透。大家工作生活压力那么大，一定要学会自我调节和放松，别让久坐酸痛发展成大问题。如果没有多余的时间和金钱出去享受，可以像我一样费个妙界在身边，它只要背上就能用两档热敷，配合三档力度，可以迅速帮助肌肉恢复活力，手头上的事情完全不会耽误，操作也非常简单，闲下来还能按腰按腿，特别实用。热销好评回购三榜第一的大牌子，质量靠谱，售后全面收运费险，现在618立减活动开始了，领券叠加优惠能拿到全年底价，有需要的朋友别错过了。

从我过去十多年金融搬砖所看到的视角，但凡和广义财政有关的行业，其就业工资都是享受经济红利的，而这不仅仅指的是体制内，还包括广义财政里的房地产业和金融业等等。实际上，房地产在本世纪前20个年头里，其行业工资都是很不错的，和它密切相关的上下游也颇有得意，而金融业更是如此。而他们所转移的利润来自于实体中的私企和居民部门，这个我在双轨制歧视的视频里已经详细解释过。国企和民企的现金流分化也不是什么新鲜事，这些年一直在加强，尤其在对民营资本的规范期间，其实对就业的负影响非常大，你可以看到有些细分行业可能一夜就消失了。由于强计划控制属性，这种一放就乱、一管就收的二极管现象其实一直没有变，经济一直在双轨制的歧视扭曲里发展，而失衡变成了一种均衡。但投射到就业上，弱势一方的利润分配很少或者倒贴，于是就业岗位的新增力量就不断承压，岗位供给变少，而求职供给增加，这种不对称就转化成了中枢向上的青年失业率，而且在不少行业我们看到他的工资还在下降。而居民部门的就业收入预期、消费者信心预期随之就会下滑，这其实挺严重的。而聚集起来的财政现金流，又在这么多年被转化成了上期视频所提及的存量债务，而这是所有人都要思考的一个关键问题。

而最近金融时报有一篇报道很深刻，他说目前国内女性是婚恋成本唯一一种投资，而是投资就会期待回报，而这个回报就需要男性买单，而关键是很多年轻男性无法在30岁之前积累起足够的买单实力，所以男性的压力很大。2024年，中国青年报一项调查显示，78.4%的受访青年表示住房压力大，其中男性和在一线城市的青年压力更大，因为一线城市的房价收入比更高，意味着年轻人获得婚恋入场资格的门槛更高，而尤其对于非土著居民而言，这就是经济和分配对年轻人所产生的影响。而年轻人推迟和拒绝婚育，又会对经济产生影响，而由于中口供给端已经对生产投资等宏观控制模式产生了变化，所以存量劳动力其实在背负更卷更长的劳动环境前行，背负的人苦不堪言，而得意的人却大言不惭的说，你看一切都很好啊。而这和我上一期视频描述低保障率的城乡居民一样，而多少人的口舌之快只是建筑在他们真实的痛楚之上，而快乐中竟然还有个别放肆的人会这么说，你能奈我何。

5月金融数据显示居民部门信贷收缩明显，前五月居民贷款新增规模创2009年以来最低。行为经济学中的最后通牒博弈理论显示，当经济主体感知分配不公时，即使牺牲自身利益也会采取消极互惠。这种现象在宏观经济中体现为居民对抗性储蓄增加与信贷需求萎靡，反映出公众对当前分配机制和信任体系的抵触情绪。

教育领域呈现高强度内卷态势。北京大学家庭追踪调查显示，2022年中国高中生周学习时间达59.7小时，超过职场人士11小时。中科院针对3万青少年的研究发现，14.8%存在抑郁风险，其中4%达到重度级别。睡眠研究显示，小学生平均睡眠仅7.56小时，中学生与高中生持续面临睡眠剥夺。这种以牺牲身心健康为代价的竞争机制，正通过代际传递加剧社会焦虑。

就业市场呈现典型二元结构分化。统计数据显示，非私营单位平均工资达私营企业1.8倍，2023年机关事业单位与国企员工仅占劳动人口的23.5%。智联招聘数据显示，近75%应届生首选体制内就业，但在7000万毕业生中仅742万人能获得相关岗位。城镇员工周工作时间从2015年45.5小时增至2023年49小时，劳动力市场呈现"越竞争越内卷"的恶性循环。

房地产价值缩水加剧财富流失。国际清算银行数据显示，2024年全国房价较2019年下降12%，城镇居民人均住房资产缩水8.52万元，超过2020-2024年人均净收入总和7.89万元。商铺资产贬值40%使人均财富蒸发3.2万元。这种资产负债表衰退效应，与青年婚育意愿降低形成负向反馈：社科院调查显示超半数大学生认为婚育非必要，51.3%的25-29岁群体保持未婚状态。

公积金数据披露口径的调整折射制度信任危机。2024年公积金年报修改披露格式，隐藏企业性质分类数据。当前仅23.5%就业人口参与公积金制度，其中近半集中于体制内单位。青年群体承受多重压力：78.4%受访者表示住房负担沉重，一线城市房价收入比高企构建婚恋门槛，教育回报率下降加剧学历贬值焦虑。当经济增长红利无法有效传导至新生代，日本式"低欲望社会"特征正在显现。政策制定者亟需重建透明可信的分配机制，避免经济主体从消极互惠转向全面抗税。
