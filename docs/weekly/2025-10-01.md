[Awesome Health](http://github.com/kakoni/awesome-healthcare) 一份高质量的医疗领域开源项目清单。适合医疗软件开发者、医院 IT 人员和医疗研究人员，可以快速找到所需的开源解决方案。覆盖了各个方面，从电子病历系统到医学影像处理，从远程医疗到机器学习，每个项目都经过精心筛选。
主要内容：

- 电子病历系统（EHR）：包含 OpenMRS、Bahmni、HospitalRun 等 20+ 个开源系统；
- 医学影像处理：涵盖 DICOM 服务器、图像查看器、3D 可视化等工具；
- 开发框架和库：提供 FHIR、HL7 等标准的实现库，支持多种编程语言；
- 医疗数据标准：整理了 FHIR、HL7、DICOM 等核心标准规范；
- 机器学习工具：收录专门用于医疗健康的 AI 工具和数据集；
- 医疗物流管理：包含供应链、资产管理等实用系统。

[Claude Code Viewer](https://github.com/d-kimuson/claude-code-viewer)
目前主流的终端 AI 编程工具 Claude Code，虽然功能强大，但是命令行操作，始终阻碍着部分技术小白上手使用。
可以看一下， 这个开源工具，它为 Claude Code 提供了一个相对功能完整的直观 Web 操作界面。
不仅支持实时查看对话日志、启动和恢复会话，轻松管理所有对话，还内置了 Git 差异对比和提交功能，方便审查和提交代码。
除此之外，还支持上传图片、PDF 和文本文件，每种格式都有专门的预览组件，以及可以设置定时任务，在达到限制恢复后自动继续执行任务。
通过 npm 全局安装或直接运行即可使用，支持 macOS 和 Linux 系统，想要使用 Claude Code 的初学者可以看下。
