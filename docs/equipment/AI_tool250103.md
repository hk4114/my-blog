# 2025 我常用的 AI 产品

> 一句话总结：用好 AI 之前得先用"好 AI"。

## 我现在是用这些

如果你不想看细节，我直接把我的订阅上来

- 日常对话：Gemini(写长文/逻辑) + Kimi(日常/手机端) + 豆包 + 偶尔 qwen + 爬虫 Grok
- 编程：Cursor + Claude Code (命令行 模型 MinMax)
- 知识管理：[notebookLM(ppt 思维导图)](https://notebooklm.google.com/) + [ima（知识库管理）](https://ima.qq.com/) + [youmind（整理碎片想法信息）](https://youmind.com/zh-CN/)
- 工作流：[coze(配合飞书)](https://www.coze.cn/) + [n8n（本地工作流）](https://n8n.io/)

之前专门写过一篇文章，里面有很多产品，但是品类太多了，其实平时用的也就那么几个，写这篇文章的目的也是总结自己平时使用的产品，以及自己是如何使用的。
在用好 AI 之前，应该多用用业界最好的 AI 产品。
如何知道现在哪类模型好呢？
除了用户评价，还可以在 [lmarena](https://lmarena.ai/zh/leaderboard) 大模型竞技场上找找，当然最重要的还是多用多体验。

## 2. 编程

目前的编程工具分两类：IDE（集成开发环境）和 Terminal（命令行工具）。

- [Cursor](https://www.cursor.com/) (IDE)
- 地位：目前的行业标准。
- 价格：Pro 版约 ¥140/月。

- [Trae](https://www.trae.ai/) (IDE)
- 地位：字节出品，Cursor 的强力竞争者，不仅免费还非常好用。
- 价格：首月 21rmb/月 后续 70rmb/月。

- Claude Code (Terminal)
- 搭配：建议配合国产模型 (Minimax/kimi k2) 的 API 使用。

## 3. 图像

专业工具（MJ/Flux）门槛太高，我目前都是用这两个：

- [即梦 (Jimeng)](https://jimeng.jianying.com/)
- 包含：文生图、文生视频。
- 费用：基础会员 ¥69/月（每日有免费积分）。

- Gemini (nano banana2)
- 费用：含在 Gemini 订阅中，或通过 API 调用 (约 ¥1/张)。每人每天 3 张限额

## 4. 效率与知识库

- [NotebookLM](https://notebooklm.google.com/) 丢进去一堆 PDF，能生成播客、摘要、思维导图。完全免费。
- [Ima](https://ima.qq.com/) 适合整理凌乱的想法。
- [Youmind](https://youmind.com/zh-CN/) 类似 notebooklm。

- 深度搜索 (DeepResearch)：
- [秘塔 AI 搜索](https://metaso.cn/)：国内搜资料首选，无广告，直达信源。
- [Perplexity](https://www.perplexity.ai/)：国外搜资料首选。
- [ithy](https://ithy.com/)

## 5. 智能体 (Agent)

- [Coze (扣子)](https://www.coze.cn/)：配合飞书使用，搭建自己的工作流机器人。
- [n8n](https://n8n.io/) 本地工作流
- [Manus](https://www.manus.ai/) / [AutoGLM](https://autoglm.zhipuai.cn/)：能够操控浏览器的智能体，帮你自动订票、填表。

## 我的最终建议

不用做囤囤鼠，可以先尝试 一个主力模型 (Gemini/ChatGPT) + 一个主力编辑器 (Cursor/Trae) + 一个顺手的搜索工具 (秘塔/Perplexity) 开始慢慢改造自己的工作流。
